### 配置 SMTP
1. 获取 SMTP 提供商的配置信息（例如 SendGrid、Mailgun、Amazon SES 等）。
2. 登录 Supabase 仪表板，进入 `Authentication` 设置。
3. 在 `SMTP Settings` 部分，填写 SMTP 提供商的配置信息：
    - `SMTP Host`：SMTP 主机地址
    - `SMTP Port`：SMTP 端口
    - `SMTP User`：SMTP 用户名
    - `SMTP Pass`：SMTP 密码
    - `SMTP Sender Name`：发送者名称
    - `SMTP Sender Email`：发送者电子邮件地址
4. 保存设置。
